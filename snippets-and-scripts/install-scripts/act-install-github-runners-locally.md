# act (Install GitHub Runners Locally)

[act](https://github.com/nektos/act) allows you to run GitHub Actions locally by using Docker.

## Installation

```bash
curl -s https://raw.githubusercontent.com/nektos/act/master/install.sh | sudo bash
```

Or install via your package manager. See the [act repository](https://github.com/nektos/act) for full documentation.
