# Vehicle Tracking

Open-source stacks for **live vehicle telemetry**, GPS tracking, and fleet dashboards.

## In this section

| Page | System |
|------|--------|
| [Open Vehicles documentation](./open-vehicles-documentation.md) | OVMS — EV/ICE remote monitor & control |
| [traccar.org](./traccar.org.md) | Traccar GPS tracking server |

## OVMS (high level)

- Open Vehicles Monitoring System — module + cellular link to your car
- SoC, temperatures, tyre pressure, charge control (vehicle-dependent)
- Open source — you own your data vs OEM telematics

## Traccar (high level)

- Self-hosted GPS tracking platform
- Supports hundreds of device protocols
- Web UI + REST API for fleet maps and history
